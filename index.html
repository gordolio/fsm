Diagram Designer

Your browser does not support
the HTML5 <canvas> element

Import from: Graphviz

Export as: PNG | SVG | LaTeX

Instructions:

  • Add a shape: double-click on the canvas
  • Add an arrow: shift-drag on the canvas
  • Resize: alt-drag the object
  • Move: drag it around
  • Delete: click it and press the delete key (not the backspace key)
  • Switch circle to accept state to rectangle: double-click on a shape to toggle
  • Type numeric subscript: put an underscore before the number (like "S_0")
  • Type greek letter: put a backslash before it (like "\beta")
